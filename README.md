# rdcq

A package implementing methods for the estimation of cell abundaces from omics data.

## Install
```R
devtools::install_github("ddiez/rdcq")
```

## Usage

Load the package and run the shiny application by typing:

```R
library(rdcq)
dcqUI()
```
